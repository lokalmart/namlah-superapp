# Game Ant Art Branch

Branch ini adalah eksplorasi tampilan Namlah Superapp versi game koloni.

## Tujuan

- Memberi alternatif visual yang lebih imajinatif untuk showcase.
- Menjaga flow Semut-ID, PIN, role, map, Gudang, Scan, dan Akun tetap sama.
- Membuktikan satu codebase bisa berganti rasa visual tanpa mengganti domain flow.

## Perubahan Rasa

- Beranda menjadi arena sarang dengan tile, resource node, unit semut berjalan, dan HUD.
- Role-ID disebut seperti class role: Explorer, Scout, Runner, Trade Post, Crafting, Hive Command, Treasury.
- Aktivitas menjadi quest card.
- Gudang Semut menjadi inventory/shop.
- Scan Jejak menjadi portal scan.

## Batas

- Tidak menjalankan server lokal.
- Tidak menambah backend.
- Tidak menambah dependency baru.
- Tidak mengganti repo utama Namlah Studio.
